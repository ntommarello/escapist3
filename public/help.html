Custom Domains

Custom domains let you host your events with a URL like "www.mywebsite.com".

First, you'll need to buy a domain from a registrar like GoDaddy, pairNIC, Register, or iWantMyName. This usually costs $10-$40 per year. Then you'll need to follow the registrar's instructions to point the domain's A-record (IP address) to 50.19.236.102

Please note that you need to change your domain's or sub-domain's A Record, not its nameservers. And, you don't need to forward or redirect anything, you only need to change the A Record.

Once you've done this, you may need to wait up to 72 hours for the change to take effect. If the domain is set up properly, you should see the Escapist home page when you visit your domain.

Creating a Facebook Application

In order to use the Facebook Connect on your custom domain, you'll need to create a Facebook application on the Facebook site. The application will allow your site and Facebook to connect to each other.

  • Login to Facebook with your usual Facebook account. Then register your site as an application on Facebook at http://developers.facebook.com/.
  • You'll then need to name your application and accept the Terms of Use.
  • Go to the Website tab for the Facebook app settings.
  • In the Site URL field, type in the URL to your site domain (ex: http://www.yoursite.com).
  • In the Site Domain field, type in your site domain (ex: yoursite.com).
  • Take note of the Application ID and Application Secret code that Facebook has generated for you, as they need to be entered into your Settings.
  • The rest of the fields are not important to the FBC module, but you can fill out some of them if you wish.
 

MailChimp
The API key is like your password. To get it, log into MailChimp, click the orange Account button at the top left of the screen, then click 'API Keys & Info'. Your API key is the long string of digits and numbers in the Key column
 

Stripe
This API keys are like your password. To get it, log into Stripe, then click
'Your Account' on the left menu. Your API key is the long string
